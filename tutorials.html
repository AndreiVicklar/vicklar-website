Driver tutorials

Vicklar Driver menu-by-menu guide.

Use these written tutorials to understand each main workflow before full screenshot and video tutorials are published. The guides are practical, driver-focused and designed for quick reference after a long shift.

Setup Cloud & Plans Logbook Checks Reports Employer sharing
1. Start clean

Set up the app first.

  • Choose your preferred app language on first run or later from Settings.
  • Add your driver name, vehicle details, office email and preferred units.
  • Open Cloud & Plans if you want backup, restore and encrypted cloud document storage on Pro.
  • Keep Dashboard quick access buttons for the menus you use most during a working day.
2. Daily workflow

Use the app during the shift.

  • Use Logbook for the working day, mileage, locations and activity segments.
  • Use Daily Checks and Weekly Checks to record vehicle condition and defects.
  • Capture fuel and expense receipts as soon as you receive them.
  • Store important document photos in Documents instead of leaving them scattered in the gallery.

Module instructions

How to use each menu.

These guides explain what each menu is for, what to check, and where users should be careful before saving or sending data.

⌂

Dashboard

  1. Use Dashboard as the daily home screen before starting work.
  2. Check the account, plan and cloud status area first.
  3. Use quick access for the menus you open most during a shift.
  4. Open All menus when you need the full Driver toolbox.
Back to menu ↑
⚙

Settings

  1. Choose language, units and driver details.
  2. Add your office email if you regularly send reports.
  3. Review quick access buttons so the dashboard matches your working day.
  4. Use disk space and cloud preferences when you change phone or storage setup.
Back to menu ↑
☁

Cloud & Plans

  1. Check your current Free, Basic or Pro status.
  2. Use Restore purchases after subscribing through Google Play.
  3. Use manual sync before uninstalling or changing phone.
  4. Use restore when moving to a replacement phone, then verify the restored records.
Back to menu ↑
↺

Restore Assistant

  1. Use this flow after login on a new phone when cloud data is available.
  2. Choose restore from cloud instead of starting fresh when you want previous records back.
  3. Keep the app open until restore finishes.
  4. After restore, check documents, logbook, checks, fuel and expenses before relying on them.
Back to menu ↑
☰

Logbook

  1. Create the daily record as close as possible to the working day.
  2. Check date, vehicle registration, mileage, locations and activity segments.
  3. Use previous-day pre-fill only as a helper, not as a final value.
  4. Review weekly and monthly summaries before exporting or sending.
Back to menu ↑
◉

Tacho slip photo scan

  1. Take a straight, clear photo with good light.
  2. Use the active slip area, not surrounding paper or old history below it.
  3. Review all 9 detected values: date, REG, KM start, start time, finish time, KM end, driving, other work and POA.
  4. Refresh or edit incorrect fields before saving.
Back to menu ↑
✓

Daily Checks

  1. Open Daily Checks before using the vehicle.
  2. Mark each item OK only after checking it.
  3. Record defects clearly and add photos where useful.
  4. Generate or send a PDF record when required by your employer or agency.
Back to menu ↑
✓

Weekly Checks

  1. Use Weekly Checks for deeper periodic vehicle records.
  2. Choose the correct saved vehicle profile before starting.
  3. Attach photos for defects or important condition evidence.
  4. Review the finished check before PDF/export.
Back to menu ↑
◎

Weekly Tyre Checks

  1. Create the correct vehicle or trailer tyre profile first.
  2. Record tread, pressure and condition values carefully.
  3. Compare against the previous check when available.
  4. Use the PDF report as supporting tyre history evidence.
Back to menu ↑
▤

Documents

  1. Add only documents you need for driver work or personal organisation.
  2. Set expiry dates so reminders are useful.
  3. Use front/back photos where the document has two sides.
  4. Free and Basic use local limits; Pro can back up encrypted cloud document copies.
Back to menu ↑
☎

Contacts

  1. Add offices, agencies, yards, planners and support numbers.
  2. Keep names clear so you can find contacts quickly after a long shift.
  3. Update old numbers instead of creating duplicates.
  4. Pro can sync contacts to cloud for restore on a new device.
Back to menu ↑
⛽

Fuel

  1. Record fuel soon after filling.
  2. Check date, vehicle, litres, cost and mileage.
  3. Attach the receipt photo if you need proof later.
  4. Keep fuel records separate from random gallery photos.
Back to menu ↑
£

Expenses

  1. Record parking, tolls, washing, parts or other work-related costs.
  2. Choose the right category and check the amount.
  3. Attach a receipt photo where possible.
  4. Review exported summaries before submitting to an employer or agency.
Back to menu ↑
⏱

Timesheet

  1. Choose the correct date range.
  2. Review imported Logbook periods before generating the timesheet.
  3. Check breaks, start/finish times and totals.
  4. Export or send only after the figures match your own records.
Back to menu ↑
PDF

Agency Pack Reports

  1. Use Agency Pack when you need a compact work report.
  2. Select the days and sections required for the agency.
  3. Review mileage, duty totals, allowances and notes.
  4. Generate the PDF only after checking the included records.
Back to menu ↑
£

Pay Calculator

  1. Create a pay profile with your hourly, day, night or allowance rules.
  2. Import timesheet data where useful.
  3. Add deductions or extras carefully.
  4. Treat the result as a helper and check it against your official payslip.
Back to menu ↑
CV

Driver CV

  1. Add licences, experience, sectors, languages and availability.
  2. Keep the profile private unless you choose to share or publish it.
  3. Use clear descriptions that employers can understand quickly.
  4. Pro can back up the CV to cloud for restore.
Back to menu ↑
⇄

Loads and Custom Loads

  1. Use Loads for planned transport work and personal load records.
  2. Add collection and delivery details clearly.
  3. Attach proof photos only where useful and allowed.
  4. Keep records tidy so you can find the job later.
Back to menu ↑
🏢

Employer Loads

  1. Use Employer Loads when connected to a participating company workspace.
  2. Follow the instructions from the planner or employer.
  3. Add collection, delivery and proof photos as required.
  4. Sync when online so the office can review shared operational records.
Back to menu ↑
🔗

Employer Access

  1. Use Employer Access only when your employer gives you a valid connection.
  2. Check the company name and connection status before sharing data.
  3. Employer-funded Pro can unlock cloud features while the seat is active.
  4. Disconnect or ask support if the company link looks wrong.
Back to menu ↑
Important note

The driver stays in control.

Vicklar Driver helps organise records and speed up repetitive administration, but drivers must still check every scanned, imported or manually entered value before relying on it.

The app does not replace legal responsibilities, employer instructions, official tachograph records or professional judgement.
